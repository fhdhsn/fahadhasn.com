Frequently Asked Questions

The app can't find my location

Make sure Location Services are enabled for ChargePK in Settings → Privacy & Security → Location Services. The app only needs "While Using the App" access.

A charger's details are wrong or missing

Charger data is community-sourced. Email the station name and the correction, and it will be fixed in the next update.

A new charging station opened — can it be added?

Yes! Send the station name, location (or Google Maps link), charger type (AC/DC), and price if known.

The journey planner shows a straight dashed line instead of a road route

That happens when road routing isn't available for a segment. The dashed line is an estimate; charger results along the corridor still work.

Reporting Bugs

When reporting a bug, it helps to include your iPhone model, iOS version, and a screenshot if possible.
